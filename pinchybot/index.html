The concept

PinchyBot is a multipurpose chat bot solo project that started in 2013 as a way to learn about the Python programming language. Three variants have been written for different chat protocols.

The Discord variant

The Discord version of PinchyBot was initially written in C# in March 2016, but later moved on to Python in order to avoid majorly re-writing most of the side functions it was packed with from the other two variants. PinchyBot relies mostly on asynchronous tasks as well.

It's a feature!

PinchyBot is loaded with over 25 user accessible commands, a list of them including descriptions are below.

Interaction messages. PinchyBot will react to certain things that are said (Though their name has to be used to avoid false reactions)

    • User commands

    When a command takes an argument, it will be listed out for those commands in brackets or curly braces. [] are for required arguments, and {} are for optional arguments, do not use the brackets or braces when you type a command argument.

    If the command you're wanting to use isn't on this page, then it likely isn't in the bot's code as each one of its commands excluding developer commands will be documented here.

    • $ping

      • Ping! Pong! As the command says, this is good for checking if PinchyBot is online and working or not.

    • $reverse [argument]

      • Reverses the order of the supplied argument, for example, reversing "Robot" would be "toboR".

    • $eightball [question] ($8ball aliases to this)

      • Get a random answer from asking a question, even though the question is merely a placeholder and has no effect on the RNG that determines the answer.

    • $hug

      • Get hugged by PinchyBot.

    • $wz {location}

      • NOTE: Because PinchyBot is relying on an API that only allows limited calls on a free plan, the weather commands ($wz, $wf, $wzadd, and $wzrm) are only enabled on whitelisted servers. If you want the weather commands enabled on your server, you must ask the bot owner through Discord, preferably on the bot's home guild (See bottom of page). If PinchyBot is in a lot of servers, especially large ones, then I likely won't add any more servers to the whitelist!

      Fetches current weather conditions for the location, argument is required if you haven't saved your location (See $wzadd and $wzrm). If you're using a location that isn't within the USA or Canada, then you must provide the country name or abbreviation itself (London, UK for example), otherwise state/province names can be used. Do not use just the city name as it can have conflicting names within the search query in some cases. Alerts, if any are present, will be shown alongside the conditions.

    • $wf {location}

      • Fetches 4 day forecast, take note of argument explaination above.

    • $wzadd [location]

      • Saves the entered location into a database under your unique ID on Discord (Even if you change your name but still be on the same account, PinchyBot will still know it's you) A search query is also run and PinchyBot will warn you if the location could not be found. If you want to change the location that was already saved, just simply issue this command with a new location.

    • $wzrm

      • Removes your location from the database

    • $pony {query}

      • Fetches a random image from derpibooru based on the query ('cute' tag is searched if argument isn't supplied). NSFW searches by default are filtered out unless allowed for the channel the command is issued in (Will implement a feature for server owners/admins to toggle this sometime soon!)

    • $dice [XdY]

      • Roll dices! X is the number of dices to roll, Y is how many sides each dice has.

    • $cmdlist

      • DMs you the link to this page.

    • $epsearch [query]

      • Search for any existing MLP:FiM episodes, outputs the season and episode number as well as the airing date in UTC (Disabled by default due to potential spoilers)

    • $epnext

      • Shows the name of the next episode as well as its airtime, may not be available in between season or in-season hiatuses. (Longer than a week between episodes. Like $epsearch, this is disabled by default)

    • $timer [time]

      • A timer that directly mentions you when the time is up, upper limit is 2 days. Argument examples: 30s, 5m, 15m30s, 2h30m, 4h15m30s

    • $ytsearch [query]

      • (May be removed as it isn't seeing much use at the moment) Search for a video on youtube.

    • $pong

      • What happens when the bot dev takes suggestions? This happens.

    • $svwhen

      • Shows when the server was created

    • $fatcount

      • Shows how many times PinchyBot has been called fat, counts are per server

    • $ship

      • Get shipped with a random currently online user on the server

    • $binary [argument]

      • Converts the supplied argument to binary, numbers are treated as integers and will have varying bit lengths (EG 12 is 1100 in binary). ASCII characters (or letters if you must) will have each of their 8 bits shown, this includes spaces.

    • $rolelist

      • DMs you the list of roles a server has. Useful for fetching their IDs

    • $candy

      • This was a meme.. right..?

    • $svstats

      • Shows how many servers PinchyBot (and the member count of each server combined) is in

    • $info

      • ($cmdlist may be replaced by this) Shows some info including an invite link to its home server and a link to add PinchyBot to a server.

    • $calc [expression]

      • PinchyBot is suddenly a calcluator, expression must follow proper Python syntax. + Add, - Subtract, can be used for negative numbers, * Multiply, / Divide, ** Exponent, ^ Bitwise XOR (Exclusive OR)

    • $dectohex [int]

      • Converts decimal to hexadecimal

    • $hextodec [hex]

      • Converts hexadecimal to decimal

    • $wat

      • In Pinchy we trust, one Wat

    • $avatar {@user}

      • Shows a user's (or yours) avatar

    Server adminstrator commands

    (Not to be confused with debug commands which will not be listed here) Commands for toggling features such as NSFW queries on Derpibooru, this feature currently a work in progress.

    The commands listed below are restricted to those with the Manage Server permission. (Server owners count as they override every permission regardless of role)

    • $enablensfw

      • Enables NSFW Derpibooru searches for the channel the command is issued in.

    • $disablensfw

      • Disables NSFW Derpibooru searches for the channel the command is issued in. Do note that NSFW searches are disabled by default.
